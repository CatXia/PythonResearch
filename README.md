## 安装
* 安装python3.8
* 安装Anaconda
* 安装PyTorch
    - conda install pytorch torchvision -c pytorch
* 安装TensorFlow
    - conda install tensorflow
* 安装NumPy
    - conda install numpy
* 安装SciPy
    - conda install scipy
* 安装MatplotLib
    - conda install matplotlib
* 安装IPython
    - conda install ipython
* 安装Pandas
    - conda install pandas
